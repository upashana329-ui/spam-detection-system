🛡️ AI-Powered Spam Detection System
This is a production-ready AI-powered spam detection system that can classify messages/emails as Spam or Not Spam (Ham) with high accuracy. The system uses multiple machine learning algorithms and features a modern, user-friendly GUI.

🎯 Key Highlights
✅ 98.39% accuracy on UCI SMS Spam Collection dataset
✅ 3 ML models with real-time model switching
✅ Modern dark-themed GUI built with PyQt5
✅ Batch processing support for CSV files
✅ Real-time predictions with confidence scores
✅ Visual analytics with confusion matrix and performance charts
✨ Features
🤖 Machine Learning
3 Powerful Models: Logistic Regression (98.39%), Random Forest (98.03%), Naive Bayes (97.22%)
TF-IDF Vectorization with n-gram support
Automatic data balancing for imbalanced datasets
Model persistence - Save/Load trained models
🎨 GUI Features
Dark Modern Theme - Easy on the eyes
Real-time Prediction - Instant results with confidence scores
Model Switching - Compare different models on the fly
Batch Processing - Upload CSV files for bulk analysis
Visual Analytics - Confusion matrix and performance charts
